# Hoop-IQ
Quiz where the user can select a decade from the NBA; 2000's, 2010's, 2020's and then test their Hoop IQ from the decade.
# Features
1. 3 Multiple Choice Questions and 2 fill in the blank Questions for each NBA Decade
2. Shows a score for total correctly answered and missed questions at the end
3. Correct Answers will NOT be shown at end of quiz. If you want the answers, there will be an option for an answer sheet for all decade questions at the start of the ran code.
