## Description

Hoop-IQ is an interactive Python quiz game that lets you test your NBA knowledge by selecting from three different decades:
- **2000s** - The Shaq & Kobe Era
- **2010s** - The Big Three & Warriors Dynasty  
- **2020s** - The Modern Era

Each decade features questions about:
- NBA Finals winners
- MVP award winners
- Scoring leaders
- Historic moments and achievements

The game includes both multiple-choice and true/false questions to challenge your basketball IQ!

## How to Play

1. Run the game:
   ```bash
   python3 nba_trivia_quiz.py
   ```

2. Select a decade (1, 2, or 3)

3. Answer 5 trivia questions about that decade

4. Get your score and see how well you know your NBA history!

5. Play again with a different decade or quit
